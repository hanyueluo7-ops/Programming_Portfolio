# Programming_Portfolio

This repository contains two programming assignments completed in Google Colab, focusing on data and spatial visualisation using Python. All datasets used are included. The first assignment explores the relationship between two variables through a customised visualisation, involving data cleaning, improved marker design, appropriate colour palettes, and clear titles and axis labels. A short markdown explanation is provided in the notebook to justify the visualisation choices.

The second assignment presents a static spatial visualisation showing the distribution of a selected variable. It applies suitable mapping techniques, data preprocessing, and enhanced visual styling, including customised legends and colour schemes to improve interpretation and accessibility. A brief markdown section in the notebook explains the design decisions and methodology.
